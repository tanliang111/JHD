Provides all things directly related to representing a structured drawing and editing it.

The most important interfaces in this package for representing a drawing are: {@link org.jhotdraw.draw.Figure}, {@link org.jhotdraw.draw.Drawing}, {@link org.jhotdraw.draw.ConnectionFigure}, {@link org.jhotdraw.draw.Connector}, {@link org.jhotdraw.draw.CompositeFigure};

The most important interfaces in this package for editing a drawing are: {@link org.jhotdraw.draw.DrawingEditor}, {@link org.jhotdraw.draw.DrawingView}, {@link org.jhotdraw.draw.Tool}, {@link org.jhotdraw.draw.Handle}.
